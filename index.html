Summary: Latitude Vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. In order to accomplish this analysis, our first task was pulling data from the OpenWeatherMap API to assemble a dataset consisting of over 500 world cities that were pulled at random. After assembling the dataset and cleaning it in Python Pandas, we used Matplotlib to create plots of various aspects of weather phenomena vs. latitude. Factors we looked at included: maximum temperature, percent cloudiness, wind speed, and humidity. There is a page that provides the source data as well as four pages for each visualization created as part of the analysis. Tethered to each analysis page are explanations and descriptions of any trends and correlations witnessed.
